## How to Use

1. Select a BYD model from the dropdown
2. Choose the year (2024-2027)
3. Optionally enter a custom price or toggle export model status
4. Click "Calculate Price" to see the breakdown
5. View final price, subsidies applied, and total savings in both THB and CAD
